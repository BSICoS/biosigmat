# `ipfmExample`

Example demonstrating IPFM-based instantaneous heart rate estimation.

## Description

This example demonstrates how to estimate instantaneous heart rate and the normalized modulating signal from beat occurrence times using the ipfm function. The process involves loading ECG beat timestamps from a fixture, constructing the IPFM spline representation, evaluating it on a uniform sampling grid, and visualizing the resulting instantaneous heart rate and modulating signal.
